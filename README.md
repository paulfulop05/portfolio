# Portfolio Website

> ⚠️ **Work in Progress** - This portfolio is currently under development and incomplete.

A modern, interactive portfolio website built with React, TypeScript, and Tailwind CSS.

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Radix UI

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```
